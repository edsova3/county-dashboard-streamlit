# 📊 Project Overview
This project analyzes county-level socioeconomic data across 5 counties in Southern Ohio and Northern Kentucky using Python

## 🚀 Live Dashboard
[view the live app](https://edsova3-county-dashboard-streamlit-app-yzuknz.streamlit.app/)

## Business Report 
[Read the full analysis](Business_Report_County_Dashboard.pdf)

## Key Insights
- No single county dominates across all economic metrics
- Jackson (OH) and Pike (OH) counties show strong employment conditions
- Greenup County (KY) has the highest educational attainment
- Ross County (OH) provides a balanced economic environment

## Tools used
- Python (pandas, Streamlit)
- Tableau
- Excel
